# AI Agent with Python and Langchain

## Description
This project implements a simple AI chatbot using Python and Langchain.

## How to Run
1. Clone the repo:
   git clone https://github.com/zeghariAyoub/AI-Agent-with-Python-and-Langchain.git
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   python Chatbot.py

## Notes
- Ensure you have Python 3.10+ installed
- Add any API keys in a `.env` file (do not commit secrets)
- This is a prototype for learning and experimentation
